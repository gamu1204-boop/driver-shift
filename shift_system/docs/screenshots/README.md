# スクリーンショット

このフォルダに以下のスクリーンショットを配置してください。

## 必要な画像

1. **main.png** - トップページ（index.html）
2. **schedule.png** - 週間スケジュール管理画面
3. **driver.png** - ドライバー登録画面
4. **course.png** - コースマスター画面

## 推奨サイズ

- 幅: 1200px 程度
- 形式: PNG

## 撮影方法

1. ブラウザでシステムを開く
2. サンプルデータを入れた状態で表示
3. ブラウザの開発者ツール（F12）→ デバイスツールバーで幅を調整
4. スクリーンショットを撮影
